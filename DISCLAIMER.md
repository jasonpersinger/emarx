# Disclaimer

EMARX is intended to bundle only public-domain texts or texts that are clearly licensed for redistribution.

Copyright status varies by jurisdiction. A work that is public domain in one country may still be protected in another. Users and distributors are responsible for confirming that redistribution is lawful in their jurisdiction.

This project should not bundle modern copyrighted translations, Marx/Engels Collected Works edition text, modern introductions, annotations, footnotes, publisher notes, or other editorial apparatus unless the relevant license clearly permits redistribution.

Each bundled work should have a matching metadata file that records:

- source URL
- source name
- author or authors
- translator or editor if known
- original publication date if known
- license or public-domain note

If a text's redistribution status is unclear, do not bundle it.
