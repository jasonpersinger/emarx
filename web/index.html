INSTALL.DOC

Run EMARX locally

Install the native Rust CLI/TUI with Cargo, then launch the reader from any terminal.

cargo install --git https://github.com/jasonpersinger/emarx
emarx
requires: Rust + Cargo try: emarx list try: emarx search "commodity fetishism"
RUNNING PROCESS

emarx --reader-demo

The browser demo keeps the CLI shape contained. The installed app opens full-screen in your terminal.

ABOUT.TXT

A reader for difficult, necessary texts

Marx is not only a name attached to politics. He was a relentless critic of work, money, property, history, and the systems that teach people to mistake exploitation for common sense. His writing can be dense, funny, furious, technical, and startlingly alive.

EMARX is built around the idea that these texts deserve a quiet, durable reading environment: no feed, no account, no recommendation engine, no decoration competing with the page. Just a terminal, a corpus, search, bookmarks, and source notes.

The goal is to make public-domain Marx and Engels texts easier to keep close at hand, while treating sources and copyright status with care. The project starts small on purpose so the library can stay auditable as it grows.
