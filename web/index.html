WHATIS.TXT

A serious reader for unruly texts

Emarx is a single executable named emarx. Run it with no arguments and it opens a full-screen terminal reader with works, sections, source notes, search, bookmarks, themes, and resume behavior.

It is built for public-domain and clearly redistributable writing by Marx, Engels, and related authors. The project starts small on purpose so the corpus can stay auditable as it grows.

The website gives you the map. The installed app is the machine.

INSTALL.DOC

Install the native terminal reader

Install from GitHub with Cargo, then launch the TUI from any terminal. Binary releases are planned; source install is the current path.
