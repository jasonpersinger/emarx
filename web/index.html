EMARX

Read Marx from the command line

EMARX is an offline-first terminal reader for Marx, Engels, and related public-domain texts. Browse works, search passages, and read in a compact command-line interface.

Install View on GitHub

install

Run EMARX locally

Install the native Rust CLI/TUI with Cargo, then launch the reader from any terminal.

bash
cargo install --git https://github.com/jasonpersinger/emarx
emarx

requires Rust and Cargo

also try emarx list · emarx search "commodity fetishism" · emarx read manifesto

about

A compact humanities reader for the terminal

EMARX bundles a starter corpus of public-domain or freely redistributable texts and provides list, read, search, random passage, source metadata, bookmark, and resume workflows without network access during normal use.

works

Included starter corpus

  • The Communist Manifesto
  • Wage-Labour and Capital
  • Value, Price and Profit
  • Critique of the Gotha Programme
  • The Eighteenth Brumaire of Louis Bonaparte
  • Theses on Feuerbach
  • Preface to A Contribution to the Critique of Political Economy
  • Capital, Volume I
EMARX Public-domain and freely redistributable source texts only. github.com/jasonpersinger/emarx
